# 文档入口（面向人类）

本目录用于存放**人类读者**（开发/使用者）需要的文档：架构、接口、工作流与 UI 设计等。

如果你是第一次接触本项目，建议阅读顺序：
1. `docs/arch/overview.md`
2. `docs/api/spec.md`
3. `docs/guides/debug_sop.md`
4. `docs/guides/workflows.md`
5. `docs/arch/frontend_design.md`

## 目录结构
- `docs/arch/`：架构与 UI/UX 设计
- `docs/api/`：接口规范（MCP 工具 + Orchestrator HTTP）
- `docs/guides/`：调试 SOP 与工作流

## coding agent 专用材料
